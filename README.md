# FoodStory 🍳

Приложение про готовку (SwiftUI + SwiftData, iOS 26). Это твой учебный pet-проект.

## Как открыть

1. Распакуй архив.
2. Открой файл **FoodStory.xcodeproj** двойным щелчком (откроется Xcode).
3. Сверху выбери симулятор (например, iPhone 16) и нажми ▶︎ (Cmd+R).
4. При первом запуске приложение само добавит 3 примера рецептов.

> ⚠️ Я собирал проект без компилятора Xcode (на сервере его нет), поэтому
> возможны мелкие правки при первом запуске — это нормально и это часть обучения.
> Если Xcode где-то подчеркнёт ошибку красным — пришли мне текст ошибки, разберём.

## Где что лежит

```
FoodStory/
├── FoodStoryApp.swift      ← точка входа, подключение базы SwiftData
├── ContentView.swift       ← нижняя навигация (5 вкладок)
│
├── Theme/
│   └── AppTheme.swift       ← ВСЕ цвета и размеры приложения (светлая «кухонная» тема)
│
├── Models/                  ← данные (что хранит приложение)
│   ├── Recipe.swift          • рецепт (главная модель)
│   ├── Ingredient.swift      • ингредиент
│   ├── Step.swift            • шаг приготовления
│   ├── Difficulty.swift      • сложность (enum)
│   ├── IngredientUnit.swift  • единицы измерения (enum)
│   ├── SampleData.swift      • примеры рецептов
│   └── PreviewSupport.swift  • данные для предпросмотра в Xcode
│
├── Cooking/
│   └── CookingSession.swift  ← «движок» режима готовки (шаги, таймер)
│
└── Views/                   ← экраны (что видит пользователь)
    ├── Home/HomeView.swift            • Главная
    ├── Recipe/
    │   ├── RecipeListView.swift        • список «Мои рецепты»
    │   ├── RecipeDetailView.swift      • детальный экран рецепта
    │   ├── AddRecipeView.swift         • форма создания рецепта
    │   └── Components/
    │       ├── RecipeCardView.swift     • карточка рецепта
    │       ├── AddIngredientSheet.swift • добавление ингредиента
    │       └── AddStepSheet.swift       • добавление шага
    ├── Search/SearchView.swift        • Поиск
    ├── Fridge/FridgeView.swift        • Холодильник (подбор по продуктам)
    ├── Profile/ProfileView.swift      • Профиль + статистика
    └── Cooking/CookingView.swift      • полноэкранный режим готовки
```

## Что уже работает (MVP)

- ✅ Список рецептов с сортировкой (дата / популярность / время / алфавит)
- ✅ Детальный экран: ингредиенты с галочками, шаги, метаданные
- ✅ Добавление рецепта (вкладки: основное / ингредиенты / шаги)
- ✅ Режим готовки: пошагово, таймеры, экран не гаснет
- ✅ Поиск по названию и ингредиентам
- ✅ Холодильник: подбор рецептов по совпадению ингредиентов
- ✅ Профиль со статистикой
- ✅ Локальное хранение (SwiftData)
- ✅ Светлая тёплая тема
