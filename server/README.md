# FoodStory — сервер-прокси к Claude

Маленький сервер, который прячет ключ нейросети. Приложение обращается к нему, а
он вызывает нейросеть. Так секретный ключ **не попадает в приложение** — это
обязательное условие для публикации в App Store.

**Поддерживаются два «мозга»:**
- **Gemini** (Google) — есть **бесплатный** тариф, ключ без карты: https://aistudio.google.com → Get API key. Задать переменную `GEMINI_API_KEY`.
- **Claude** (Anthropic) — платный: https://console.anthropic.com. Задать `ANTHROPIC_API_KEY`.

Провайдер выбирается автоматически: если задан `GEMINI_API_KEY` — используется
Gemini; иначе Claude. Можно задать явно: `AI_PROVIDER=gemini` или `claude`.

## Что внутри

- `POST /suggest` — принимает `{ "products": ["яйца","сыр"], "note": "быстро" }`,
  возвращает `{ "text": "…рецепт…" }`.
- `GET /health` — проверка: `{ "ok": true, "configured": true }`.

## Запуск локально (для проверки)

```sh
cd server
npm install
ANTHROPIC_API_KEY=sk-ant-ВАШ_КЛЮЧ npm start
```

Проверка: открой http://localhost:8787/health — должно быть `"configured": true`.

> ⚠️ С телефона по `http://` iOS блокирует запросы (нужен `https`). Для быстрой
> проверки с телефона подними туннель, например:
> `npx cloudflared tunnel --url http://localhost:8787` — он даст временный
> `https://…`-адрес, его и впиши в Настройках приложения.

## Деплой (постоянный https-адрес)

Подходит любой хостинг Node. Проще всего — бесплатные варианты вроде Render или
Railway:

1. Залей папку `server/` в отдельный git-репозиторий (или весь проект).
2. Создай новый **Web Service**, укажи корень `server/`, команда запуска `npm start`.
3. В переменных окружения задай:
   - `ANTHROPIC_API_KEY` — твой ключ (получить: https://console.anthropic.com);
   - по желанию `APP_SHARED_TOKEN` — любой секрет (тот же впиши в приложении);
   - по желанию `CLAUDE_MODEL=claude-haiku-4-5` — модель подешевле.
4. После деплоя получишь адрес вида `https://foodstory.onrender.com`.

## Подключение в приложении

Приложение → **Профиль → Настройки → AI-помощник (сервер)**:

- **Адрес сервера** — твой `https://…`;
- **Токен** — если задавал `APP_SHARED_TOKEN`.

Теперь на экране «Помощник» кнопка **«Придумать новый рецепт (AI)»** будет
генерировать рецепты через Claude. Пока адрес не задан — помощник работает
офлайн (подбор из твоих рецептов).

## Переменные окружения

| Переменная         | Обязательна | Описание                                  |
|--------------------|-------------|-------------------------------------------|
| `ANTHROPIC_API_KEY`| да          | Ключ Anthropic                            |
| `CLAUDE_MODEL`     | нет         | Модель (по умолчанию `claude-opus-4-8`)   |
| `APP_SHARED_TOKEN` | нет         | Общий секрет приложения (заголовок `x-app-token`) |
| `PORT`             | нет         | Порт (по умолчанию 8787)                  |
